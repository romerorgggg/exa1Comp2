📋 Descripción

Este proyecto implementa una simulación de dinámica molecular de N partículas esféricas confinadas en una caja rectangular. Las partículas interactúan mediante colisiones elásticas entre sí y con las paredes de la caja. El proyecto está implementado en C++ utilizando Programación Orientada a Objetos (POO) y sigue las mejores prácticas del Google C++ Style Guide.

🎯 Objetivos

  • Simular el movimiento de N partículas en una caja 2D
  • Implementar colisiones elásticas partícula-partícula y partícula-pared
  • Comparar métodos de integración numérica (Euler y Velocity-Verlet)
  • Analizar distribución de velocidades (Maxwell-Boltzmann)
  • Estudiar conservación de energía
  • Analizar presión mediante conteo de colisiones

📊 Experimentos Implementados

1. Gas Diluido

  • N = 25 partículas
  • Radio = 0.1, velocidad inicial ≤ 1.0
  • Estudia trayectorias y difusión en sistema poco denso

2. Gas Denso

  • N = 100 partículas
  • Radio = 0.15, velocidad inicial ≤ 2.0
  • Estudia sistema con alta densidad de colisiones

3. Comparación Euler vs Verlet

  • N = 50 partículas
  • Compara conservación de energía entre métodos
  • Demuestra superioridad de Velocity-Verlet

4. Análisis de Presión

  • Varía tamaño de caja (5×5, 10×10, 15×15)
  • Cuenta colisiones con paredes
  • Relaciona con presión P ∝ nm⟨v²⟩/L²

5. Distribución de Velocidades

  • N = 200 partículas
  • Genera histogramas de |v|
  • Compara con distribución Maxwell-Boltzmann 2D

🔬 Métodos Numéricos

Integración de Euler

x(t+dt) = x(t) + vₓ(t)·dt
y(t+dt) = y(t) + vᵧ(t)·dt

Velocity-Verlet (recomendado)

x(t+dt) = x(t) + v(t)·dt + 0.5·a(t)·dt²
v(t+dt) = v(t) + a(t)·dt

Colisiones Elásticas

Para dos partículas de masas iguales:

Impulso = 2m₂·(v_rel · n̂)/(m₁ + m₂)
v₁' = v₁ + impulso·n̂
v₂' = v₂ - impulso·n̂

🎨 Estilo de Código

El código sigue el Google C++ Style Guide:

  • Nombres de clases: PascalCase (ej: Bola, Caja)
  • Nombres de métodos: PascalCase (ej: MoverseEuler())
  • Miembros privados: snake_case_ con guion bajo final (ej: x_, velocidad_)
  • Constantes: kConstanteName
  • Comentarios Doxygen en todos los métodos públicos
