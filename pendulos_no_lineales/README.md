# Proyecto: Péndulos Acoplados con Interacción No Lineal

**Autor:** Romero y Garzon
**Materia:** Computación II  
**Tema:** Dinámica no lineal y modos normales de oscilación  
**Método numérico:** Runge–Kutta 4 (por defecto) y Euler–Cromer (opcional)

---

##  Descripción general

Este programa simula **dos péndulos acoplados** con interacción **no lineal** y permite analizar:
 Transferencia de energía entre los péndulos
 Modos normales de oscilación (en fase y en oposición)
 Regímenes de **resonancia** y **caos**
 Representaciones en el espacio fase y espectros de frecuencia

El sistema es interactivo: desde el ejecutable puedes seleccionar el tipo de simulación, cambiar parámetros físicos, o regenerar las gráficas con Python.
