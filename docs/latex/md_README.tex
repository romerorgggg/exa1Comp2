\#\# Descripción del Proyecto

Este proyecto implementa la simulación numérica de \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}dos péndulos acoplados con amortiguamiento, forzamiento externo y una interacción no lineal cuadrática\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}. El sistema se resuelve usando \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Runge-\/\+Kutta de 4º orden (RK4)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} o \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Euler-\/\+Cromer\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, permitiendo al usuario configurar todos los parámetros de forma interactiva.

El software, desarrollado en C++17, permite explorar fenómenos como la \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}transferencia de energía\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}sincronización\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}resonancia\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} y \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}caos\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}.
